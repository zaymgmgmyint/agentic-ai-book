# Agentic AI အခမဲ့လမ်းညွှန်

မြန်မာ Developer များအတွက် Agent, Tool, MCP, RAG, Coding Agent နှင့် DevOps Automation လက်တွေ့အခြေခံစာအုပ်။

ဤ `book/` folder သည် uploaded manuscript draft များနှင့် GitHub repo-backed case study လေးခုကို တစ်နေရာတည်းတွင် ပြန်စုစည်းထားသော v0.1 normalized manuscript ဖြစ်သည်။ မူရင်း draft များကို မဖျက်၊ မပြင်၊ မအစားထိုးထားပါ။ ဤနေရာရှိဖိုင်များသည် public release အတွက် ပြန်ရေးထားသော copy များသာဖြစ်သည်။

ဤစာအုပ်သည် production manual မဟုတ်ပါ။ Framework documentation ကို မြန်မာလိုပြန်ဆိုထားသော report လည်းမဟုတ်ပါ။ Agentic AI ကို မြန်မာ Developer တစ်ယောက်က ဘယ်လိုစဉ်းစားရမလဲ၊ Tool တစ်ခု၏အန္တရာယ်ကို ဘယ်လိုမြင်ရမလဲ၊ Agent Loop နှင့် Harness ကို ဘယ်လိုခွဲသိရမလဲ ဆိုသည့် အခြေခံလမ်းပြစာအုပ်ဖြစ်သည်။

ဖတ်ရန်အစမှာ `chapters/00-license.md` နှင့် `chapters/00-preface.md` ဖြစ်သည်။ Chapter 12 မှ 15 အထိသည် မဖြုတ်မရသော repo-backed case study လေးခုဖြစ်သည်။

1. [`P-2`](https://github.com/htooayelwinict/P-2) — state boundary နှင့် zero-trust handoff
2. [`travis-2`](https://github.com/htooayelwinict/travis-2) — controlled runtime
3. [`bot / BrowserSurfer Bot`](https://github.com/htooayelwinict/bot) — browser tool agent security
4. [`allthebest / appv22`](https://github.com/htooayelwinict/allthebest) — emerging runtime recovery lab
